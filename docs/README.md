## Decisão crítica

O frontend React pode ser publicado no GitHub Pages. O backend Node.js, autenticação e uploads devem ser executados fora do GitHub Pages.

## Segurança

Nenhuma credencial real foi incluída nesta documentação. A senha administrativa inicial deve ser configurada como segredo no backend e alterada antes do lançamento.
